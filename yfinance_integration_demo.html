🔧 API-Endpunkte

1. Aktie per Ticker hinzufügen

POST /stocks/add-by-ticker Content-Type: application/json { "ticker_symbol": "AAPL", "watchlist_id": 1 }

2. Aktiensuche

GET /stocks/search/AAPL Response: { "found": true, "stock": { "ticker": "AAPL", "name": "Apple Inc.", "sector": "Technology", "industry": "Consumer Electronics", "country": "US", "current_price": 253.81, "pe_ratio": 25.43 } }

3. Marktdaten aktualisieren

POST /stocks/{stock_id}/update-market-data Response: { "message": "Market data updated successfully", "data": { "current_price": 253.81, "pe_ratio": 25.43, "timestamp": "2025-09-30T12:00:00Z" } }

🚀 So verwenden Sie das neue Feature

Frontend starten:
cd frontend && npm start
Backend starten:
uvicorn backend.app.main:app --reload
Watchlist erstellen und "Aktie hinzufügen" klicken
"📈 Per Ticker-Symbol" wählen und z.B. "AAPL" eingeben
Suchen & Hinzufügen: Aktie wird automatisch mit allen Daten hinzugefügt!
